It is a tool that helps protect the privacy of your data by keeping sensitive information anonymous.
It detects Personally Identifiable Information (PII) in Microsoft Word documents and replaces it with realistic, randomly generated data while preserving the document's structure and formatting.

#Technologies
-Python
-pyhton-docx
-spaCy
-Faker
-Regular Expression (Regex)

#Flow
1> Load your document file.
2> Extract the key data and name it while preserving the format.
3> Detect PII with Regex and sapCy.
4> Generate and replace with fake values generated by Faker.
5> Save the redacted data as a New File.

#PII
- Full Names
- Email Addresses
- Phone Numbers
- Company Names
- Physical Addresses
- IP Addresses
- Credit Card Numbers
- Security and Official Document Numbers
- Date of Birth
